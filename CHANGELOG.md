# Changelog

All notable changes to **ENV Manager** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/) and this project adheres to Semantic Versioning.

## [0.0.4] - 2026-07-18

### Changed

- Improved README documentation.
- Added installation instructions for both VS Code Marketplace and Open VSX.
- Added project roadmap and FAQ.
- Improved usage documentation.
- Updated support section with Ko-fi information.
- Refined extension description and Marketplace presentation.

---

## [0.0.3] - 2026-06-20

### Added

- 🎉 Initial public release.
- Multi-profile support for `.env` files.
- Secret masking for sensitive environment variables.
- Visual editor for managing environment variables.
- Search and filter support.
- Export `.env.example`.
- Save changes directly to `.env` files.
- Published to Visual Studio Marketplace.
- Published to Open VSX.
